Kurzanleitung zur aktuellen Version

Software

Nur um PDF zu erstellen (unter Debian) (texlive-full ist overkill, aber einfach): 
# apt-get install git gitweb texlive-full
Editieren: in Emacs gibt es Vorschau für alle Formeln und Skizzen: 
# apt-get install emacs auctex preview-latex-style

Runterladen und PDF erstellen

$ git clone git://github.com/pzorin/ipho-aufgabensammlung.git
$ cd ipho-aufgabensammlung
$ ./topdf

Lokales Editieren

Lokalen Arbeitszweig erstellen und in diesen wechseln: 
$ cd ipho-aufgabensammlung
$ git branch mylocalbranch
$ git checkout mylocalbranch
Änderungen in Git eingeben und einen Patch erstellen: 
$ git commit -a
$ git format-patch -M -B master
Den Patch könnt ihr mir per e-Mail senden. Wer mehr machen möchte sollte sich überlegen ein eigenes Repository zu starten.
