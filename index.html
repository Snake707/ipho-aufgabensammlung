Kurzanleitung zur aktuellen Version

Absolut notwendige Pakete unter Debian installieren (ist zwar overkill, dafür aber einfach): 
# apt-get install git gitweb texlive-full

Runterladen und PDF erstellen

$ git clone git://github.com/pzorin/ipho-aufgabensammlung.git
$ cd ipho-aufgabensammlung
$ ./topdf

Lokales Editieren

Folgendes wurde vermutlich schon ausgeführt: 
$ git clone git://github.com/pzorin/ipho-aufgabensammlung.git
$ cd ipho-aufgabensammlung
Nun sollte ein lokaler Arbeitszweig erstellt und in diesen gewechselt werden: 
$ git branch mylocalbranch
$ git checkout mylocalbranch
Es wird empfohlen Emacs zu benutzen, da man damit Vorschau für alle Formeln und Skizzen bekommt: 
# apt-get install emacs-gtk auctex preview-latex-style
*hackety hack* Nun ist es Zeit die Änderungen Git zu überlassen und einen Patch zu erstellen: 
$ git commit -a
$ git format-patch -M -B master
Den Patch könnt ihr mir per e-Mail senden. Wer eventuell mehr machen möchte sollte sich überlegen ein eigenes Repository zu starten.
